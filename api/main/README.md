# Главный модуль Битрикс `main`

## Основные классы

- CMain - главный класс страницы
- CDBResult - класс результата выполнения запроса
- CFile - класс для работы с файлами и изображениями
- CBitrixComponent - оболочка компонента
- CBitrixComponentTemplate - оболочка шаблона компонента
- CAgent - класс для работы с функциями-агентами
- CDatabase - класс для работы с базой данной
